## 🧑‍💻 Getting Started

To run this project locally:

### 1. Clone the repository

```bash
git clone https://github.com/your-username/3d-portfolio.git
cd 3d-portfolio
```

### 2.Install dependencies

```bash
npm install
```
### 3.Start the development server

```bash
npm run dev
```
